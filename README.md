# Flash CLI

## Install & Build from source

```sh
pnpm install && pnpm compile
pnpm link --global
flash # ready to be used
```

Supported frameworks:

- Next.js
- Nuxt.js
- Lume (Deno)
- static

Supported providers:

- IPFS
  - nft.storage
  - Estuary (WIP)
  - Filebase (WIP)
  - Pinata (WIP)

Supported protocols:

- IPFS
- Arweave (Maybe/WIP)
- Filecoin (WIP)

Supported Serverless/Web3:

- Gelato (WIP)
- Taubyte (Maybe/WIP)
